filterList

Description

Uses an input box to filter an unordered list in real-time.

Required HTML

<form>
  <input type="text" id="filter">
  <ul>
    <li>Item A</li>
    <li>Item B</li>
    <li>Item C</li>
  </ul>
</form>

Required JavaScript

$('#filter').filterList();
